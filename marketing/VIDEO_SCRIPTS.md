## 📋 Production notes

### Shooting checklist

- [ ] iPhone stabilized (small tripod or phone grip)
- [ ] Airplane mode ON before recording — visual proof is key
- [ ] Record in 4K @ 30fps, vertical 9:16
- [ ] Screen recording via iOS Control Center (enable first)
- [ ] For B-roll: 1-2 second shots, keep it tight
- [ ] Shoot 3 takes of each shot, pick best

### Editing checklist (CapCut)

- [ ] Import all clips, drag to timeline in order
- [ ] Add caption track — CN on top, EN below, white text with subtle drop shadow
- [ ] Use Apple-style font: Noto Sans TC (CN) + Inter/SF Pro (EN)
- [ ] Music at 30-40% volume (speech would drown it anyway — no speech here)
- [ ] Brand colors for end card: Deep Navy `#0B1438` background, Cyan `#5EE2D7` accent
- [ ] Export: 1080×1920, H.264, max bitrate
