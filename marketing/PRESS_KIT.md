## 📸 Press Release (EN)

### FOR IMMEDIATE RELEASE

**A privacy-first translator that actually works offline — without collecting your words.**

*Taipei, Taiwan — 2026-04-21* — Independent developer [@mina0717](https://twitter.com/mina0717) today released **Offline Translator v1.1** on the App Store, a free iOS app that combines four translation tools — text, voice, photo (OCR), and a 170,000-word offline dictionary — into one interface. Everything runs on-device. No servers. No analytics. No ads.

"Most translator apps silently ship your most sensitive moments — travel medical phrases, contract clauses, personal messages — to a foreign server," said the developer. "I built this because I travel a lot and I don't want my words in anyone else's database. If the airplane mode still works, the privacy is real."

**v1.1 introduces six new capabilities:**
- **Vocabulary Notebook** — save any translation; export to CSV
- **Share Extension** — translate selected text from Safari, Notes, Messages
- **Hold-to-Speak** — long-press to record, release to translate, with haptic feedback
- **Siri Shortcut** — "Translate clipboard" via voice
- **TipKit onboarding** — four contextual tips for first-time users
- **Dedicated dark-mode palette** — hand-tuned for low-light reading

Offline Translator is built on Apple's native `Translation` framework (iOS 17.4+), with MLKit as a fallback for earlier iOS versions. Speech recognition runs entirely on-device via `SFSpeechRecognizer`, and OCR uses Apple's `Vision` framework.

The full source code is MIT-licensed and available on GitHub: <https://github.com/mina0717/offline-translator-ios>.

## ❓ FAQ for journalists

**Q: How can you afford to run this for free with no ads?**
A: There's nothing to pay for. No servers, no analytics, no tracking — translation happens on the user's iPhone. The only cost is my time.

**Q: Where does the 170,000-word dictionary come from?**
A: Curated open-source dictionaries merged and cleaned into Core Data. Full attribution in the repo's `CREDITS.md`.

**Q: Does Apple's Translation framework actually run offline?**
A: Yes — after the first language-pack download, translation is fully offline. You can verify by enabling airplane mode.

**Q: Why did you open-source it?**
A: So users don't have to trust me — they can read the code and confirm nothing leaves the phone.

**Q: What's next for v1.2?**
A: Home Screen Widget, iPad Split View with dual-column translation, iCloud sync for the vocabulary notebook, and SharePlay for live translated captions in calls.

**Q: Is it available on Android?**
A: Not planned. Android's on-device translation story is different; I'd want to design from scratch rather than port.

**Q: Who's behind this?**
A: One person. @mina0717 on Twitter/X. Based in Taiwan. Independent — no company, no VC, no team.
