## 🚀 Launch Day 速查

打開 `LAUNCH_COPY.md`，按順序執行：

1. **T-3 暖身**：IG / Threads 連發 3 天預告
2. **Day 0 00:01 PT**：Product Hunt 發布
3. **Day 0 09:00 PT**：X thread / LinkedIn / Hacker News / Reddit 接力發文
4. **Day 0 全天**：IG / Threads 5 卡 carousel + pinned maker comment
5. **Day 1-7**：回覆 PH / HN 評論 + 逐日貼 UGC
